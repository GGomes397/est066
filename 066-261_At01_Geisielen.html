A distruição usada será Gumbell \[F(x;\mu, \beta) = e^{-e^{-(x-\mu)/\beta}}\]

A função quantílica é dada por :

\[ u= e^{-e^{-(x-\mu)/\beta}}\] \[\ln(u)=-e^{-(x-\mu)/\beta}\] \[-\ln(u)=e^{-(x-\mu)/\beta}\] \[\ln(-\ln(u))=-\frac{x-\mu}{\beta}\] \[-\ln(-\ln(u))=\frac{x-\mu}{\beta}\] \[x=\mu-ln(-ln(u))\beta \]

Para definir a função de gerador de números aleatórios fizemos :

mu <- 0
b <- 1

gera_gum <- function(mu,b,n){
  u <- runif(n,min = 0, max = 1)
  x <- mu - b * log(-log(u))
  return (x)
}

alpha <- 0.9


rejt <- function(n, alpha){
  
  x <- numeric(n)  # vetor para guardar resultados
  i <- 1
  
  while(i <= n){
    
    u <- runif(1)
    
    if(u < alpha){   # aceita somente se u < alpha
      x[i] <- u
      i <- i + 1
    }
    
  }
  
  return(x)
}

z2 <- rejt(1e3, alpha)

Questão 7

Sejam \(X_1, X_2 \sim N(0,1)\) variáveis aleatórias independentes e identicamente distribuídas (i.i.d.). Suas funções de densidade de probabilidade (f.d.p.) são dadas por: \[f_{X_1}(x_1) = \frac{1}{\sqrt{2\pi}} e^{-\frac{x_1^2}{2}} \quad \text{e} \quad f_{X_2}(x_2) = \frac{1}{\sqrt{2\pi}} e^{-\frac{x_2^2}{2}}\] Como \(X_1\) e \(X_2\) são independentes, a f.d.p. conjunta é o produto das densidades marginais:

\[f_{X_1, X_2}(x_1, x_2) = f_{X_1}(x_1) \cdot f_{X_2}(x_2) = \frac{1}{2\pi} e^{-\frac{x_1^2 + x_2^2}{2}}\] Desejamos encontrar a densidade da variável \(Y = \frac{X_1}{X_2}\). Utilizando a fórmula da densidade da razão para duas variáveis aleatórias, temos: \[f_Y(y) = \int_{-\infty}^{\infty} |x_2| f_{X_1, X_2}(y x_2, x_2) \, dx_2\] Substituindo a densidade conjunta na integral, onde \(x_1\) é substituído por \(y x_2\): \[f_Y(y) = \int_{-\infty}^{\infty} |x_2| \frac{1}{2\pi} e^{-\frac{(yx_2)^2 + x_2^2}{2}} \, dx_2\]

Colocando \(x_2^2\) em evidência no expoente:

\[f_Y(y) = \frac{1}{2\pi} \int_{-\infty}^{\infty} |x_2| e^{-\frac{x_2^2(y^2 + 1)}{2}} \, dx_2\]

Dada a simetria da função em relação a \(x_2\), podemos integrar de \(0\) a \(\infty\) e multiplicar o resultado por 2: \[f_Y(y) = \frac{2}{2\pi} \int_{0}^{\infty} x_2 e^{-\frac{x_2^2(y^2 + 1)}{2}} \, dx_2\]

Para resolver esta integral, aplicamos a substituição: \[u = \frac{x_2^2(y^2 + 1)}{2} \implies du = x_2(y^2 + 1) \, dx_2 \implies x_2 \, dx_2 = \frac{du}{y^2 + 1}\]

Substituindo os termos na integral: \[f_Y(y) = \frac{1}{\pi} \cdot \frac{1}{y^2 + 1} \int_{0}^{\infty} e^{-u} \, du\]

Visto que \(\int_{0}^{\infty} e^{-u} \, du = 1\), obtemos a f.d.p. final: \[f_Y(y) = \frac{1}{\pi(1 + y^2)}\]

Esta expressão corresponde exatamente à função de densidade de probabilidade de uma distribuição da Cauchy padrão.

gera_cauchy_normais <- function(n) {
 
  x1 <- rnorm(n)
  x2 <- rnorm(n)
  return(x1 / x2)
}
